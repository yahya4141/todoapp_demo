**ToDo List** is a web-based task management application developed to practice DOM manipulation and event handling using **jQuery**. The application stores data in the browser's **LocalStorage**, ensuring that tasks persist even after the page is refreshed.

### Built With
* **HTML5 & CSS3**
* **Bootstrap 5** (Modals, Cards, Grid System)
* **JavaScript (jQuery 3.7.1)**
* **Ionicons**

### Features
* **Create & Edit Tasks:** Add new tasks with a title, category (Shopping, Homework, etc.), and due date. Edit existing tasks via a modal.
* **Smart Sorting:** Sort tasks alphabetically (A-Z, Z-A) or by status (Completed First / Incomplete First).
* **Search:** Real-time filtering by task name.
* **Task Management:** Mark tasks as "Done" or delete them individually.
* **Bulk Action:** "Delete All" button with a confirmation warning modal.
* **Data Persistence:** Automatically saves all changes to `localStorage`.
* **Validation:** Prevents duplicate task names and ensures required fields are filled.

### How to Run
1. Clone or download this repository.
2. Open `todo_1.html` in your web browser.
3. No server installation is required.
